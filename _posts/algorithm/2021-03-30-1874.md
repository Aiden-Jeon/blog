problem: 1874  
link: https://www.acmicpc.net/problem/1874  

Input: empty stack 1 // n
Output: array <a1, a2, ...> ai = (push, pop) if cannot return NO

스택 수열 동작 예시

source code: 
```python
import sys


N = int(sys.stdin.readline())
stack = []
result = []
s = 1
for _ in range(N):
    j = int(sys.stdin.readline())
    while True:
        if s <= j:
            stack.append(s)
            result.append("+")
            s += 1
        else:
            p = stack.pop()
            if j == p:
                result.append("-")
            else:
                result.append("NO")
            break
    if result[-1] == "NO":
        break

if result[-1] == "NO":
    print("NO")
else:
    for r in result:
        print(r)
```
